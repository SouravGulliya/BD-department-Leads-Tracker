# Business Development Dashboard 

## Overview

A dynamic and interactive **Business Development Dashboard** designed to track business growth, lead conversion, onboarding progress, store performance, revenue analysis, and operational efficiency.

This dashboard provides real-time insights into:

* Lead funnel performance
* Store onboarding progress
* Revenue vs projections
* Rental efficiency
* Location-wise execution tracking
* Funnel leakage analysis
* Trend analysis for FTS, LTS, and Drops

## Key Features 

| Module | Features |
|---|---|
| **BD Overview** | Total FTS Tracking, Total LTS Monitoring, Owner-Owned Store Analysis, Live Store Monitoring, Open Lead Status, Conversion Ratios |
| **Execution Speed Monitoring** | Average BD Time, Onboarding Speed, Open Lead Aging, Drop Lead Analysis |
| **Funnel Leakage Analysis** | Lead Drop Tracking, BD Drop Analysis, LTS Drop Monitoring, Owner-Owned Lead Leakage |
| **Revenue Intelligence** | Projected vs Actual Revenue, Dine-In & Delivery Performance, Store-wise Deviation Analysis, Rental vs Net Revenue Tracking |
| **Trend Analytics** | FTS Trends, LTS Trends, Drop Trends, Monthly Performance Tracking |
